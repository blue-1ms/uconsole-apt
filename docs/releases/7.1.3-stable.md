# uConsole kernel 7.1.3 stable

This is the stable alias for the exact hardware-passed `0.1.0-candidate.16` kernel bytes.

- Runtime ABI: `7.1.3-1003-uconsole`
- Package version: `7.1.3-1003.1.uconsole.1`
- Kernel bundle SHA-256:
  `59530b18a9e9dcc692bf3b7c5af62ac678ec6e6c7d16d5543478a696655c93ca`
- Current kernel: `7.1.3-1003-uconsole`
- N-1 fallback: `7.1.3-1001-uconsole`

The individual `.deb` assets are the same bytes published by the signed stable APT channel. The
original immutable bundle remains attached to `0.1.0-candidate.16`; this stable alias does not
rebuild or replace it.

If this repository is not already configured, follow the
[first-install instructions](https://github.com/blue-1ms/uconsole-apt#first-installation).
Then install through the signed repository:

```bash
sudo apt-get update
sudo apt-get install --yes uconsole-platform
sudo uconsole-kernel-policy-validate
sudo apt-get install --yes uconsole-kernel uconsole-plymouth-theme
sudo reboot
```

Kernel installation continues through initramfs, `flash-kernel`, and `piboot-try`; do not write
boot files manually.
